# 🚀 Task Manager Web App

A simple and efficient web-based Task Management application built using **HTML, CSS, and JavaScript**.

This project allows users to manage their daily tasks with deadlines and persistent storage using LocalStorage.

---

## 📌 Features

* ➕ Add new tasks with:

  * Title
  * Description
  * Due Date
* ✅ Mark tasks as Complete / Incomplete
* 🗑 Delete tasks
* 💾 Persistent storage using Browser LocalStorage
* 🔄 Tasks remain saved even after page refresh
* ⏰ Deadline tracking functionality

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Browser LocalStorage API
